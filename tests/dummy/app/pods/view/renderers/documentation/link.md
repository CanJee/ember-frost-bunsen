## link

This renderer provides an anchor tag/link for a given address.

### Properties

#### label

```json
{
  "label": "Bar",
  "model": "foo",
  "renderer": {
    "name": "link"
  }
}
```

#### renderer.label

Change what text is shown for the link.

```json
{
  "model": "foo",
  "renderer": {
    "label": "Bar",
    "name": "link"
  }
}
```
