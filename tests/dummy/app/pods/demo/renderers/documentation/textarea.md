This renderer provides a textarea input.

### Properties

#### label

```json
{
  "label": "Bar",
  "model": "foo"
}
```

#### placeholder

```json
{
  "model": "foo",
  "placeholder": "Bar"
}
```

#### properties.cols

```json
{
  "model": "foo",
  "properties": {
    "cols": 4
  }
}
```

#### properties.rows

```json
{
  "model": "foo",
  "properties": {
    "rows": 4
  }
}
```
