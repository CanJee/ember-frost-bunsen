The first piece of bunsen is the model.

The model is a JSON Schema compliant schema. Example:

```javascript
{
    type: 'object',
    properties: {
        foo: {
            type: 'string'
        },
        bar: {
            type: 'string'
        }
    }
    required: ['foo', 'bar']
}
```
